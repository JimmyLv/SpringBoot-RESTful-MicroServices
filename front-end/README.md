## front-end:

### prepare dev tools

`npm install -g yo bower grunt-cli gulp`

`yo angular`

### route

`app.js`

### view

`*.html`

### data

`controllers/*.js` 

### services

fetch data from back-end

### Build & development

Run `grunt` for building and `grunt serve` for preview.

### Testing

Running `grunt test` will run the unit tests with karma.
